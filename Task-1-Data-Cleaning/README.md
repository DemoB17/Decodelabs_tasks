Data Cleaning in Excel

Dataset Overview

This project focuses on cleaning and preparing an e-commerce dataset using Microsoft Excel. The dataset contains transaction-related information, including order details, customer information, sales values, payment methods, and coupon usage. The goal was to improve data quality and ensure the dataset was ready for accurate analysis and reporting.

Data Quality Issues Identified

During the initial review of the dataset, the following issues were identified:

- Missing values in the Coupon column.
- Inconsistent date formatting that could affect time-based analysis.
- Data quality issues that required standardization before analysis.

Data Cleaning Steps Performed

The following data cleaning activities were carried out in Microsoft Excel:

1. Date Formatting
   
   - Standardized the date column to a consistent date format.
   - Ensured all records were recognized correctly as dates for accurate sorting, filtering, and time-series analysis.

2. Handling Missing Coupon Values
   
   - Identified blank cells in the Coupon column.
   - Replaced missing values with "Free Shipping" to maintain consistency and avoid null values during analysis.

3. Data Validation
   
   - Reviewed the dataset after cleaning to ensure consistency and accuracy across records.
   - Verified that all changes were applied correctly before proceeding with analysis.

Final Cleaned Dataset

The dataset was successfully cleaned and prepared for analysis. Date values were standardized, missing coupon entries were handled appropriately, and the data was validated for consistency. The resulting dataset is reliable and ready for exploratory data analysis and business reporting.
