Task 2: Exploratory Data Analysis (EDA)

Overview

This project focuses on exploring the cleaned e-commerce dataset to uncover trends, patterns, and insights that support business decision-making.

Objective

To analyze customer behavior, sales performance, and product trends using Excel-based analysis.

Key Areas of Analysis

- Sales performance trends
- Customer purchasing behavior
- Product/category insights
- Payment and coupon usage patterns

Key Insights

- Identified top-performing products and categories
- Observed sales trends over time
- Analyzed customer purchase behavior patterns
- Evaluated the impact of coupons and discounts

Tools Used

- Microsoft Excel (Pivot Tables, Charts, Filters)

Final Outcome

The analysis provided meaningful business insights that can support strategic decisions.
