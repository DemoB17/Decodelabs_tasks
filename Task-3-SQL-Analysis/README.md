Task 3: SQL Business Analysis

Overview

This project uses SQL to analyze an e-commerce dataset and answer key business questions.

Objective

To extract insights from data using SQL queries and support business decision-making.

Business Questions Answered

- What are the total sales and revenue trends?
- Who are the top customers by revenue?
- What is the most used payment method?
- How do coupons affect sales?
- What are the monthly and yearly sales trends?

Data Preparation

Basic data cleaning and formatting were applied before analysis to ensure accuracy.

Tools Used

- SQL
- Database Management System

Key Insights

- Identified top revenue-generating customers
- Determined best-performing payment methods
- Found seasonal sales patterns
- Evaluated coupon effectiveness on sales

Final Outcome

The SQL analysis provided actionable insights for business decision-making and performance tracking.
