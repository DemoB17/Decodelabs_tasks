DecodeLabs Data Analytics Portfolio

Overview

This repository contains a complete data analytics project completed as part of the DecodeLabs program. The project demonstrates practical skills in data cleaning, exploratory data analysis, and SQL-based business analysis.

The goal of this project is to transform raw e-commerce data into meaningful business insights to support decision-making.

---

Project Structure

📊 Task 1: Data Cleaning (Excel)

- Cleaned raw dataset using Excel
- Standardized date formats for consistency
- Handled missing values in the Coupon column by filling with "Free Shipping"
- Produced a clean dataset ready for analysis

---

📈 Task 2: Exploratory Data Analysis (EDA)

- Performed data exploration using Excel tools
- Used pivot tables and charts to identify trends
- Analyzed customer behavior and sales performance
- Generated key business insights from the dataset

---

🧠 Task 3: SQL Business Analysis

- Wrote SQL queries to answer business questions
- Analyzed revenue, customers, and payment methods
- Identified top-performing customers and sales patterns
- Generated insights to support business decisions
